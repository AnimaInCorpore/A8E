A slightly updated 5200 BIOS seen in later consoles with only two controller ports. The change in this BIOS is to add a small change to detect and block startup of games that are only intended to run in PAL regions. This unfortunately shifts a large amount of code in the startup screen setup, which breaks a few games which directly jump into or copy this code.
