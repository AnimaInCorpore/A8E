## F.3 400/800 firmware

The 400/800 firmware consists of 10K of OS ROM. This is commonly stored as a single file containing contiguous contents of $D800-FFFF for emulation purposes, but this is a composite of the three ROM chips on the personality board.

The most common OS-B ROM image available, originally distributed with the Xformer emulator. Not often known is that this is actually a modified ROM image. The two changes relative to original OS-B are to hard limit the RAM detection to 52K ($D000) and to disable initialization of the PIA data direction registers.

The difference between the modified Xformer OS-B and the original OS-B is mostly unnoticeable for the majority of software, but there are rare cases. One is Cavern of Mars, which breaks on the Xformer OS-B if the two are also run on an emulator that doesn't properly emulate the correct PIA post-reset register state for the DDRA register.
