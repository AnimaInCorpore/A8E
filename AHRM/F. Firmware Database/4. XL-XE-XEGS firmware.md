## F.4 XL/XE/XEGS firmware

XL/XE class machines have on-board firmware for the OS, self-test routines, and internal BASIC. The OS and self-test are stored together in a 128kbit ROM chip, mapped primarily by the MMU to $C000-FFFF with the shadowed $D000-D7FF region remapped to $5000-57FF when the self-test ROM is enabled.

Main OS seen in 1200XL machines. Changes over OS-B include adaptation for NTSC/PAL timing, new keyboard and relocatable device handler features listed in the XL Addendum, fixes to the math pack, and removal of support for the right cartridge port only present on the 800.

This image is a composite of two 64kbit ROM chips on the 1200XL motherboard.

Updated OS for XE machines. Notably updates the self-test to remove the 1200XL-only function keys and add support for testing the extended memory in 130XE computers in the self test (though with serious bugs).

There is another bug fix in this release to swap two instructions in SIO to fix a rare command frame corruption bug, usually manifesting as an occasional timeout in disk or printer operations.

Last official version of the OS, updated specifically for new features in the XEGS - namely starting Missile Command and support for running with a detached keyboard. Otherwise this OS version is mostly interchangeable with v2 and software compatibility is very nearly the same.

This image doesn't correspond to a real ROM chip; it's sliced from the 32K image in the single combined ROM chip that the XEGS uses to match the earlier XL/XE OS ROM format.
