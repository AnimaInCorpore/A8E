## F.1 Introduction

## Signatures

All entries listed below have several signatures using commonly available checksum and hashing algorithms. The MD5, SHA1, and SHA256 algorithms are standard cryptographic hashing algorithms. The listed CRC32 checksums are using the algorithm used in many file formats, notably the ZIP file format.

Ways to compute these signatures include:

- Checking the listed CRC32 on the files in a ZIP archive.
- The md5sum, sha1sum, and sha256sum command-line tools.
- The 7-Zip command line tool, 7z.exe, can compute CRC32, SHA1, and SHA256 hashes. The -scrc option selects the algorithm.
- On Windows, certutil.exe -hashfile can compute MD5, SHA1, and SHA256 hashes.
