The original version of Missile Command, released on physical cartridges. The reason this version is notable is that it is not the version in the XEGS. The original release hardcodes the address of the keyboard table in OS-B and thus keyboard options on the title screen don't work on XL/XE/XEGS machines.

The version of Missile Command embedded within the XEGS' 32K combined ROM. It differs from the original cartridge release in that it has been fixed to use the address of the keyboard table in the XEGS OS ROM instead (and consequently would not work properly on a 400/800, if extracted to run on one).
