## 10.25 Speedy XF disk drive

Speedy XF is add-on that modifies an XF551 disk drive to be similar to a Speedy 1050, adding a new CPU, firmware, and RAM.

## Controller

The Speedy XF changes the XF551 more drastically than most disk drive modifications. It replaces the 8048 controller and 4K firmware with a 1MHz 65C02, 6532 RIOT, 32K of RAM, and 32K of firmware ROM. The resulting hardware is much more similar to a Speedy 1050 than the XF551, with mainly the drive mechanism being preserved. Table 83 shows the memory map.

## Firmware

A 32K ROM supplies the firmware, of which 24K is visible in the address map. The top 8K is used for the main drive firmware, while the other 16K is composed of two utility banks.

Although not used by the stock firmware, the Speedy XF actually supports a 64K ROM, with A15 bank switchable by RIOT PB5.

## Floppy drive controller

The FDC in the Speedy XF is the same WD1772 from the base XF551. It still runs at 8.333MHz to compensate for the 300 RPM drive mechanism, but no longer runs synchronously with the 65C02 controller, which runs at 1MHz like 1050-based drives.

An oddity is that the Speedy XF uses discrete logic to transform the 1772's status register bit 7 from Motor On to Not Ready, to match the 2793/2797 in the 1050. This is done by buffering the status register read and substituting bit 7 with the output from a drive latch switch.

## RAM-Input/Output-Timer

A 6532 RIOT supports all of the I/O connections that are normally handled by the I/O ports built into the 8048. The connections to it are arranged to mimic the RIOT connections in the 1050, with the main difference being the head stepping signals being direction/step rather than directly driving the stepper motor phases.

## Track display

Writes to addresses $4000 and $4001 controls the digits shown in the optional two-digit track number display. $4001 controls the left digit, while $4000 controls the right digit. Bits 0 through 6 of each byte control the segments of each 7-segment display, in the standard segment A-G order: top, top-right, bottom-right, bottom, bottom-left, top-left, center.

## Control latch

Writes to $4004 are captured into a control latch that drives miscellaneous drive functions. Bit 4, when set, selects side 2.
