## 15.2 Errata

## Atari Home Computer System, Hardware Manual

There are a couple of versions of this manual. The notes below pertain to the 1982 revision.

## II.27 Serial Mode Control table

The In Rate entry for the 110 mode is incorrect, saying Chan 2 instead of Chan 4. The serial input clock is always channel 4 or external. The text in the Comments column is correct.

## III.18 SKSTAT Register

The Serial Data Input Over-run and Keyboard Over-run bits are swapped. POKEY signals a keyboard overrun in bit 6 and a serial input overrun in bit 5.

## III.22 ALLPOT Register

Fast pot scan mode alone can't be used to read pot inputs through ALLPOT; starting the pot scan is required regardless. Enabling fast pot scan mode does turn off the capacitor dump transistors, but that isn't sufficient as there is a separate set of gates that force off the pot inputs outside of scan that isn't controlled by slow/fast pot scan.

## III.23 PENH Register

Any joystick trigger can latch PENH/PENV on most computer models, but on the 400 specifically only the joystick trigger on port 4 can activate the light pen input.

## Colleen Manual Audio and Serial Port Block Diagram

There are two issues in the middle of the diagram, where there is a switch selecting between channel 1 (chan 1) and channel 2 (chan 2) clock output for two-tone mode. The first issue is that the switch's polarity is backwards as shown; the legend indicates that the switches are displayed in position for 0 input, but channel 1 is used in two-tone mode when the output is a 1 (as indicated correctly on page II.26).

The second issue is that this connection is not a full switch in the hardware. It is missing an AND gate on the channel 2 side, so while channel 1's clock pulses are only used when a 1 bit is being output, channel 2's clock pulses always trigger a resync and toggle the serial output.
